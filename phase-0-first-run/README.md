# Phase 0 — First Build & First Run

## Overview

Phase 0 was the first full iteration of the build — sourcing every component, assembling the bike from scratch, soldering all connections, wiring the controller, and getting it running for the first time. The goal was a rideable proof-of-concept using a mix of purpose-built and budget generic parts before committing to higher-end components.

## Build Process

### Assembly

All major components were sourced and assembled in this phase. The frame was built up from bare, with forks, wheel, crankset, brakes, and cockpit (bars, stem, grips, throttle, display) all installed from scratch.

To lift and suspend the bike during assembly — with no stand capable of holding the weight — a **5-ton ratchet strap anchored to the ceiling** was used. This let the bike hang freely so both wheels were off the ground and all sides were accessible.

### Wiring & Soldering

All high-current connectors were soldered by hand. The controller was wired up including:
- Battery main leads
- Phase wires to motor
- Throttle signal
- Display/UKC1 harness
- Brake cut-off sensors (Shimano MT501)

The NBPower 72V 30Ah pack came with its own BMS and required matching connectors soldered to the controller harness.

### First Run

With everything wired and assembled, the bike was powered up and taken for its first run. This was the proof-of-concept moment — validating that the electrical system, drivetrain, and chassis all worked together before moving into refinement phases.

---

## Lessons & Notes

- Budget generic drivetrain parts were used intentionally at this stage — knowing they'd likely be replaced once the platform was proven
- The ratchet strap ceiling rig worked well as a makeshift suspension stand and became a recurring tool throughout the build
- The 750 lb/in rear coil is stiff for the weight — worth revisiting in later phases
