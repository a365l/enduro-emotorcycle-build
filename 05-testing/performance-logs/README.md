# Performance Logs

Dated ride/test logs with recorded data.

## Naming Convention

`YYYY-MM-DD_[description].md`

Example: `2025-07-01_first-full-trail-run.md`

## Template

```markdown
# [Date] — [Description]

## Conditions
- Location:
- Terrain:
- Temp (°C):
- Rider weight (kg):

## Battery
- Start voltage (V):
- End voltage (V):
- Estimated Wh used:

## Performance
- Distance (km):
- Max speed (mph):
- Avg speed (mph):
- Notable behaviour:

## Controller
- Any faults / beep codes:
- Controller temp at end (°C):

## Notes
```
